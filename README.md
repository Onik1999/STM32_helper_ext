# STM32 CMake Helper

STM32 CMake Helper initializes STM32 CMake projects automatically.

## Features

- Creates build/Debug and build/Release
- Generates tasks.json
- Generates launch.json for Cortex-Debug
- Generates .clangd
- Detects MCU from .ioc
- Detects project name from CMakeLists.txt

## Usage

1. Open STM32 CMake project folder
2. Open Command Palette
3. Run: STM32: Initialize Project

## Requirements

- CMake installed
- OpenOCD installed
- Cortex-Debug extension installed
